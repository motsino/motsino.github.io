All you need to know about the fetch api in JavaScript

The Fetch API is a simpler, easy-to-use version of XMLHttpRequest to consume resources asynchronously. SYNTAX: fetch("URL") is the default method to receive data. Fetch always returns, what we call a "Promise". Fetch always succeeds or gets a response unless there's some sort of error such as DNS lookup failure. Since fetch() returns a promise, we can use the catch method of the promise to intercept any error occurring during the execution of the request, and the processing is done in the then callbacks.
